# AI Code Assistant

## Project Overview
AI Code Assistant is a desktop application developed using Python and Tkinter.  
It helps users ask coding-related questions and get AI-generated answers.  
It also includes a basic debugging assistant for common programming errors.

## Features
- Ask coding questions
- AI-generated coding solutions
- Debug common errors
- User-friendly GUI
- Fast response using AI API

## Technologies Used
- Python
- Tkinter
- Groq API / Gemini API

## Modules Used
- tkinter
- groq / google-generativeai

## Project Structure
code_assistant/
│── gui.py
│── menu.py
│── README.md

## How It Works
1. User enters a coding question.
2. Application sends the question to AI API.
3. AI generates the response.
4. Response is displayed in output box.

## Debugging System
Supports:
- Division by zero
- Index error
- Syntax error

## Installation
Install dependencies:

pip install groq

or

pip install google-generativeai

## Run Project
python gui.py
